读书杂谈

2013/9/12

为什么你要读书？

人作为一个动物，见识的狭隘和局限性都是我们终其一生都是无法避免的事实。可是人究竟如何去避免自己的狭隘和偏见呢？
我始终觉得一个人成长的高度是和他的视野息息相关的。
王总曾经说过：”互联网会改变一切，一切没有被互联网改变的都会被互联网所改变。“
我觉得我就是那个被互联网这个大潮所改变的人，如果没有互联网，如果没有在网上认识那么多人，我永远都不会知道书是多么的好的一样东西。怎么会知道柳比歇夫，怎么会知道王小波，怎么会知道冯唐，怎么会知道和遇见那么多人，过着我未曾想过的生活，做着我未曾想过的事情？

人总是想过不一样的生活，有着不一样的体验。所以从秦始皇到汉武帝，每个人都想长生不老。可是长生不老真的可以让你过的更快乐吗？我不这样觉得。人往往就会犯本末倒置的错误，就是把不重要的东西认为是重要的，而重要的都错过了。所以就有了遗憾这个词

读书确实是一个很令人舒适的体验，当你沉浸在作者的描绘里的时候，你就像是过了另外一种生活一样。事实上，我个人受王小波的影响很大，我也一直都觉得人一生不在于成就多大的事业，做多牛逼的事情，而是要有趣，一直能做有趣的事情，不过这真是一件很难的事情呢，尤其是当你在一个没有安全感的国度的时候。读书的有趣之处在于我可以见到各种各样的价值观和世界观，各种各样有趣的事情，而只需要几个小时的时间和几十块钱。即使它是一个被包装过的，精致的价值观，我都喜欢。

罗素说：“参差多态乃是幸福本源。”恩，我想了想，大概就是这个样子。

而且从获取知识的角度上来说，网上通过Google搜索的信息都是比较零散的，不够系统。不系统的一个坏处就是，如果你出了问题，永远都不知道这是从哪儿出现的。

读什么书？

那么，读什么书呢？
我个人读书非常非常杂，我觉得读书是个很私人的事情，没有什么书是一个人必读的，就像永远没有哪个姑娘注定要当我老婆一样，不过要有姑娘这么说，我大概会尝试着改变一下我的看法：）。“男读王小波，女读周国平”这句话也不知道是谁说的，怎么说呢，我很讨厌这句话。
读书还是看个人喜好，因为我这个人兴趣比较广泛，读过的书就包括：散文，杂文，小说，科技，心理学，计算机，古文，易经，佛经，等等……
我觉得读书和交朋友做事情一样，你会在不经意的阅读过程中不断的筛选、过滤出，你觉得对自己有用的书，你喜欢的书，并一直读下去，那就够了。
我现在回顾了一下自己从大学期间过来的读书历程，虽然很杂，大致是这样：
励志鸡汤书——心理学——时间管理——计算机书籍——批判性思维——逻辑学——计算机书籍
这里你就能看出我的成长的轨迹了，我个人在大学的时候有过很严重的心理问题，我是后来拼命的读心理学的书加上自己每天不停的否定和重新认识自己，纠正自己的思维才度过那段低谷期的，现在回想起来，才意识到我的成长是和书是息息相关的，某种程度上来说，是读书救了我。
因为有过那段经历，所以我才会不厌其烦的买书，有的人不理解我啊，说我买这么多书怎么能看的完啊， 可是我现在已经养成了习惯，几天不读书就会感觉极度的焦虑，而且读书速度上也已经非常快了，关于读书速度上，我会在下一章讲。

我分享下我找书的方法吧：
这个是我大概一年前在知乎上写的一篇如何找书的方法：
如何找到好书？有什么技巧或建议？
就不重复写了，现在还有几点要补充一下。

根据豆瓣友邻找好书这个我已经很少用了，因为自己已经有了一些自己的读书习惯和方向，所以对于别人推荐的书也是比较有选择的了，久而久之当自己想了解一些东西的时候，自己会找一个系列的书来看，比如王总最近连续发了四条微薄说《了不起的盖茨比》我就下了一本，准备这个周末把它看完 ：）
靠谱的作者写的文章一般都很靠谱。比如我读凯文凯利的《失控》，就觉得这是本奇书，然后看他写的另外一本《科技想要什么》说到熵增熵减的地方，不禁拍手叫大叫：“写的真他妈的好！”。说实话，有时候国内的很多书我都不敢看。被畅销书害惨过，一朝被蛇咬，十年怕井绳。
论坛大多数也不太靠谱了，但高质量的博客质量还是很高，现在微信的公共帐号也火起来了，但更新时间一短，质量就无法保证，我想可能是出发点不一样吧
怎么读书？

分享下我自己的读书方法，我读书都是分类读的，世界是变化的，除了上厕所大解，人生都很难长时间的用一个姿势做一件事，因为这样会让人很不爽。
所以，读书也是这样。我读过一本书《超级快速阅读》，建议大家都去看看。
这样，我们得明白我们读书的目的是为了什么。

读小说：
对于我来说，这是属于快速阅读的范畴内的，目的是快速，弄明白情节，体会文章的意境，当然写的很好的地方我也会仔细的看，读《百年孤独》里面很多魔幻的描写，就觉得真的非常非常有意思，不得不佩服作者的想象力，于是就把那写地方看了好几遍。
读杂文：
大部分杂文是没什么营养的，不过有的人写的就很好，遇到有着深刻思考的文章，我都会读的比较慢，仔细想想为什么这么写，而且经常中断，因为各个章目之间没什么太大的联系，有时候会看到和自己作者写的和自己有过的想法非常类似就会总有一种隔着手机抱着作者大哭一场的冲动啊。
专业书（知识类）：
先看一遍文章的目录，第一遍快速浏览全文，这样就大致知道这个知识体系了，也就是大致知道这个知识的架构是什么样子，所以就不会读了后文忘了前文这种情况了。然后第二遍的时候再按章节读，然后敲一些代码，不懂的就略过。第三遍再来看全文，这样起来，三遍的时候就对着本书的认识就非常深入了。
个人的几点感受

以前看王阳明的书，王阳明的中心思想叫做：知行合一
他说一件事情，你认为自己明白了，其实并不算真正的明白，只有你真正做到，才能算明白。
到了公司，真正工作了，我才算是明白这个道理，但我觉得还不算晚。
读书是一件有意思的事情，做IT也是件有意思的事情，和人交流也是一件很有意思的事情，我们不能顾此失彼，但话说回来，我们的确是需要在几者之间做一个权衡的，人终其一生也只能做好有限的几个事情呢。

Raspberri pi 打造监控显示器

2013/9/6

硬件的配置

将SD卡装入树莓派，给树莓派插上网线和电源，直接连接HDMI接口即可（VGA需要转接头），第一次配置需要在USB上插入键盘和鼠标进行配置。

树莓派的初始配置

expand – 将根分区扩展到整张SD卡，因为整个Image才400多兆，但是现在的SD卡基本都是几个G的，除非SD卡有其他用途，一般建议选择这项，这样可以有足够多的空间来安装各种程序。

overscan – 可以扩充或者缩小屏幕的设置，除非一启动就发现显示的内容能刚好填满整个电视的画面。大部分情况下是不能填满或者超过，如果这样就要enable这个，然后在系统启动之后，更改config.txt中的overscanleft、overscanright、overscantop和overscanbottom的内容，如果显示内容超出屏幕的范围（显示不全），就要设置这些值为正值，如果是1080p，一般设置为48。如果显示的内容不能填满屏幕，则设置这些值为负值，如果是1080p，一般设置为-48。具体可以更加显示的结果，以16为步长进行调整。

configure_keyboard – 这个很重要，前面的屏幕选默认值：Generic 105-key (Intl) PC，但在Keyboard layout:时，显示出来的都是English（UK）的，要选择Other，然后在里面选择English（US），否则会出现键盘的一些符号不对或者对调，比如引号”和@符号对调，#号变成英镑符号等等。之后的两个屏幕都选择默认值就可以了，到了：Use Control+Alt+Backspace to terminate the X server? 时，选择Yes，表示用这个可以终止X Server，当整个X-Window死掉的时候可以用。

change_pass – 默认的用户名是pi，密码是raspberry，一般登录时不需要输入，但是用ssh远程连接时要用到这个用户名和密码，这里可以更改密码。

changelocale – 更改语言设置。在Locales to be generated: 中，选择enUS.UTF-8和zh_CN.UTF-8。在Default locale for the
system environment:中，选择enUS.UTF-8（等启动完机器，装完中文字体，再改回zhCN.UTF-8，否则第一次启动会出现方块）。

changetimezone – 更改时区，这个很重要，因为树莓派没有内部时钟，是通过网络获取的时间，如果设错时区，那么时间就不正确了，选择Asia –
Shanghai，没错是Shanghai，木有Beijing，这是Unix的传统。缩写是CST，不知道是China Shanghai Time还是China Standard Time。
ssh 是否激活sshd服务，应该选择激活，这是当界面死掉后唯一进入机器的通道（如果Kernel没死的话），可以找另外一部机器，用putty或者其他ssh的工具连接到这部机器上，用pi这个用户登录，至少可以实现安全重启。
bootbehaviour – 设置启动时启动图形界面，正常肯定是Yes。
重启后，将进入LXDE的X-Window。

显示器调整

只有VGA显示器的显示
这个则需要有HDMI转接口，HDMI线和有电源的HDMI转VGA转接口

分辨率信息的调整需要参考一下这个地址：分辨率调整

如若需要强制显示分辨率，修改/boot/config.txt 即可，修改完毕重启一下就ok了，当然，你得先记住你的ip,不然分辨率配置错误，进不了桌面可就惨了。

以下是我的配置：

hdmi_group=1

hdmi_mode=16

hdmi_group=2

hdmi_mode=82

hdmiignoreedid=0xa5000080
HDMI显示器
连接HDMI不需要修改配置，树莓派会自动读取显示器上的配置自适应屏幕。

需要注意的
因为HDMI的显示是读取显示器而自动适应自己的分辨率的，如果显示器在树莓派打开的时候是关闭的，而又没有配置相关的文件，树莓派会自动显示800X480的分辨率*

关闭树莓派显示器的自动关闭

可以参考一下：这里

再次打开终端，修改一下X—windows的配置文件：

vim /etc/lightdm/lightdm.conf
然后修改文件添加如下信息：

[SeatDefaults]
xserver-command=X -s 0 -dpms
这是把树莓派自动关闭显示器的时间改为0

注：树莓派本身是没有自动休眠的，想想它的耗能就知道了嘛，所谓的休眠，只是定时关闭显示器而已

安装远程桌面VNC并配置开机启动

安装VNC可以参考：这里

sudo apt-get install x11vnc
然后next安装，一般貌似就已经是自动开机启动了，如果没有开机启动，可以自己配置一下。

开机启动参考：这里

然后在桌面上添加X11VNC的图标：

进入Desktop文件夹：

cd ~Desktop
编辑一个名为 X11vnc.desktop的文件

vim x11vnc.desktop
然后把以下内容粘贴进去：

[Desktop Entry]
Encoding=UTF-8
Type=Application
Name=X11VNC
Comment=
Exec=x11vnc -forever -usepw -display :0 -ultrafilexfer
StartupNotify=false
Terminal=false
Hidden=false
然后保存，再编辑在/etc/xdg/lxsession/LXDE/autostart往里面添加一行

X11vnc.desktop
ok,我们再重启就好了。

安装Unclutter

设置无线网络

编辑配置文件：

sudo nano /etc/network/interfaces
修改为：

auto lo

iface lo inet loopback
iface eth0 inet dhcp

allow-hotplug wlan0
auto wlan0
iface wlan0 inet dhcp
wpa-roam /etc/wpasupplicant/wpasupplicant.conf
然后我们再根据我们自己的无线情况编辑：/etc/wpasupplicant/wpasupplicant.conf

network={
ssid=“SSID-GOES-HERE”
proto=RSN
key_mgmt=WPA-PSK
pairwise=CCMP TKIP
group=CCMP TKIP
psk=“WIFI-PASSWORD-GOES-HERE”
}
自动休眠的处理：

首先我们查看一下电源管理：

cat /sys/module/8192cu/parameters/rtwpowermgnt
结果是0，代表关闭。结果是1，代表最小。结果是2，代表最大。 当我们要关闭电源管理，需要

sudo vim /etc/modprobe.d/8192cu.conf

Disable power management

options 8192cu rtwpowermgnt=0
重启，配置Ok.

让畅销书去死

2013/7/6

我个人非常讨厌市场上泛滥的畅销书籍，在我看来，大多数的畅销书都是无营养且浪费时间的读物.。

比起读这种书，我觉得吐槽一下更能体现一下我的人生价值。

不信你可以去看看当当上的十大畅销书籍，不管是什么时候的畅销书。绝大多数都充斥了教人如何成功怎么赚钱的成功学，教人如何忧伤如何矫情的《小时代》，还有看似充满睿智理性的话语中隐藏着“投其所好”的所谓的人生哲理的书。

为什么我讨厌这些书？因为我曾经深受其害。

曾经我为自己读的学校之差而感到而痛苦，而烦躁，而焦虑。夜以继日的读这些成功学的书，矫情的书，让自己读起来“非常舒适的书”，可是读完以后，往往都会充满了一种情绪发泄的快感，但是快感不是快乐，总是无法持久。就像一

夜情永远都很难发展成婚姻一样，这期间的差距，岂止十万八千里？那时候我发现我做的事情越来越差，忍耐力越来越差，也越来越焦虑，甚至都不想读书了。

后来读了刘未鹏的《暗时间》和李笑来的《和时间做朋友》才让我有所改变，我认为读书带来的快乐不是它通过取悦你而体会到的，而是当书上的某个观点与我曾经的某个想法不谋而合的时候，或者当某个观点引发了我的思考的时候，或者我发现作者书里写的东西有一些毛病的时候而体会到的。

而还有许多书，属于畅销书一类，但还是有一些独特思考和价值的，这些书其实并不怎么值得去深读。

我们公司产品经理的书架上的很多书我都读过，譬如《金字塔原理》，《长尾理论》，《失控》啊都是介绍方法论和提升个人视野的很好的书。

不过我觉得还是多读点比较能够提升思考水平的书比较好，虽然这些书可能看起来没什么用，但真的是能够让人受用一生。

我讨厌说什么什么必读书之类的话，因为没有什么书是必读的，就像不是每个人都要走别人的人生轨迹一样。

不过你要是读了王小波的《沉默的大多数》的话，至少我觉得不会后悔。

——人活着就应该有趣

让畅销书去死。

进程阻塞

2013/5/29

自己经常会处在一种所谓的“进程阻塞”的状态下，就是明明知道自己想要做什么，知道怎么去做，但就是不想做。

当我在尝试简单的编程和写作的时候，我总会有种沉浸在其中很舒适的感觉。可是当我遇到点困难，或者准备做什么很难的事情，我的缺点就被暴露出来了。

我在计划的时候，总是想当然的把计划想象的过于简单，或者是说把实现的过程想象的非常理想。可是，事实上，事情的过程并不如我想象的那样流畅。可以这样说，大多数的时候，我的预期是大大短于我实际的工作量的。说的通俗点，就是“眼高手低”。

如果有时候问题并不是如我开始想象的那样简单，我就会想到放弃。可是，当事情变的比我想象的简单程度要稍微困难一点的时候，矛盾出来了，因为这个时候，我就会倾向于放弃。我总是无意识中把事情想象成非常困难，然后拖延，以至于最后放弃。拖延的过程其实也令我很痛苦，就像是计算机的一个进程一样，是占着内存的，而我在做其他事情的时候，总是会想着这件事。导致这件事情也做不好，别的事情也做不好。可怕的是，长此以往，我就积累了很坏的这个做事情的习惯。哪怕是这件事情我能做出来，我尝试了感觉有一点困难，就不会去做。

之前自己没有意识到这个问题的可怕之处，现在在做事情的过程中逐渐发现自己有这种逃避问题的倾向。经常会在做事情的时候陷入选择的陷阱：这件事做还是不做？做有哪些好处？不做有哪些坏处？决断会让我忍受一定程度上的“痛苦”，然后我会倾向于不去选择。可是这件事却总是困扰着我，让我没有办法轻松的去做事情。然后我又不得不去选择，从而陷入了选择困难的死循环中。

其实说实话，这个逻辑很简单。一件事情在这里，你只有两个选择

一、做 二、不做 ，没有现在不做明天做，没有明天不做后天做。事实上，逃避困难和自己的问题就是逃避成长，选择在自己的舒适区学习和工作固然会让你有安全感，但是这种安全感不过是“水中花，镜中月”而已。

有时候被带入了某个环境的时候，自己可能会失去某些方面判断问题的能力。这就是所谓的“当局者迷”么？呵呵。GTD的核心理论就是“Getting Things Done”，优先级很重要，但是优先级不能作为拖延的借口。按照GTD来安排自己的工作和学习，最重要的一点还是正确的设置好优先级。

还有，道理想清楚了其实就很简单。

老大经常在争论的时候会说的一句话，逻辑在哪里？我学习到的解决自己这个缺点的逻辑就是：

做事情 1、明确：做还是不做？ 2、优先级高？还是低？3、去做

列出所有自己能想到会发生的错误和问题，逐条剖析，然后去搞定它们。快速决断，不要犹豫。如果选择对了，那皆大欢喜。如果选择错了，事情还可以重来，而且如果你善于思考，下次就不会去犯同样的错误。但不去选择，连犯错的宝贵的机会都没有。

你丢失掉的，是宝贵的时间。

一些反思

2013/5/16

性格缺点:

回想起大学四年，我做每一件事几乎都是以失败告终。我思考了很久，终于发现这是我一种非常严重的害怕面对事实和喜欢逃避的性格缺点。

做飞思卡尔的时候老师叫我重构代码，我就会经常跑回宿舍几天不见影子，调车的时候遇到困难我总会想到放弃，躲到一边看小说。考试总是因为害怕复习而干脆裸考，而给自己找借口，美其名曰:不想复习。其实真正原因是自己害怕复习。口号喊的震天响，准备考研准备了几天最后又最后放弃，在南京工作了几天却又最后跑掉（不过，这个可能是我做的最正确的一个决定）。以前我觉得这些事情产生的原因是拖延症，现在我才发现拖延症的本质上就是害怕面对事实的一种懦弱的外在表现。
这些挫折都是因为我心智不够成熟而在成长中所付出的代价。

关于个性:

在知乎上看到一句话:“有个性的人很多，牛人都很有个性，但是有个性不代表你就是牛人。”深有所感。我在我和许多同学身上看到了类似:”我一定要有个性，这样我一定会（至少）看起来很牛”。的将有个性和很牛划起等号的潜意识。我们的为了个性而个性，其实在别人看来，只不过是一场拙劣的表演而已。我不喜欢假装有个性而到处复制粘贴别人语录的人，反而我更喜欢那些不刻意标新立异，随大流，而是仅仅凭借着自己的思考而独立发表意见和做出决定的，有着自己一套判断事物准则的人。

生活方式:

我的体质一向很好，这个东西又没有什么具体的量化标准，不过我体质的确是越来越差了，以前我熬夜参加比赛，三天只睡了七个小时，睡一觉一点事情都没有。而现在呢，熬夜一晚，第二天各种腰酸背痛，这都是因为不规律的生活习惯所导致的。

为人处事与自信:

老师曾经跟我说过，我的缺点是虚荣心非常强，而我遇见别人的时候也总是喜欢找出别人的缺点来挖苦。现在想来，虚荣心也好，优越感也罢，其实都是为了通过打压别人来借以抬高自己。罪魁祸首还是那源于自己内心深处的不自信感，而无法正确的评价自己和他人。
因为这个，我遇见人总是一副苦大仇深的模样，只懂得发现别人的弱点并加以挖苦而不会发现别人的闪光点来予以赞美。确实也因此少了很多朋友。

独立思考和想法
现在觉得被人灌输价值观是一件很可怕的事情，因为价值观如果不是独立思考而产生出来的就没有多大的意义可言。而简单的教别人道理是一件很愚蠢的事情，因为人的观点是自己的，强加给别人的观点，别人并不一定会接受，那就是做没有价值的事情，那就是很愚蠢的行为了。以前我喜欢做后一种事情，现在想来，我只需要分享自己的观点和经历就好，不同的人自然会有基于自己理解，何乐而不为呢？ 性格缺点:

我在北京的醉生梦死

2013/4/28

我在北京醉生梦死

出了公司，突然想起来没吃晚饭。然后去找旺哥拿行李，顺便吃个饭。现在我在公司做事情经常会做着做着，时间就过去了，而我终于到达了哪种自己很喜欢的那种”只做一件事“的状态，就觉得很开心。不过还是很感谢季总问我的问题，不然我可能还在南京继续纠结要不要再读书呢。

其实呢我觉得每个人都有着自己的爱好，有的人喜欢玩，有的人喜欢吃，而还有的人就是喜欢折腾所谓的理想，也没什么高下之分。说到底，每个人的一生也不过几十年光阴，不管你多么牛逼，一生你怎么潇洒的过，你最后都会成为大地上的一撮土，死了还不如一坨屎有价值。所以，自己要做什么，自己想做什么，别人的看法有多重要，自己也应该算清楚。

我在公司被问到是安徽来的总是会被认为是科大毕业的，我也不太想介绍我的母校，所以就直接说是个师范类的二本，周围的牛逼的学校毕业来的同学太多了，让我很有压力。所以我每天比别人早起来一个小时，比别人晚回来一个小时，就是希望能够弥补一下自己学习能力和基础与别人之间的差距吧。
我对这个本科学校从来就没有期望过，所以也就不会失望。以前我觉得自己一生都被毁了，现在看来似乎也不尽然，所以同学们不用为自己的学校太过介怀，给自己胡乱贴标签，我现在周围工作的不也大多数是名校毕业的么？只要努力，什么时候都不晚。

不知道怎么就突然喜欢上了醉生梦死这个词，可能和东邪西毒这部电影有关。我对它的理解就是时而清醒时而麻木，受伤了不知道疼痛的状态是生，而活在梦里，永远都是天堂，那就是死。我有时候真的分不清自己的状态，是清醒还是糊涂，平静还是浮躁，唯一明白的就是。我在北京醉生梦死。

做普通的人，干普通的事

2013/4/24

曾经一直都觉得自己是个精力超常的人，自己是个聪明绝顶的人，是个酒量超天的人，是个做什么事情都能做到最好的人。

但是后来的一次次的失败都在提醒我，我的想法是错的。一切都只不过是自己的幻觉罢了，而我却一直选择性的忽视掉了。而从没考上好的大学开始，我就一味的把自己的失败归结于自己的不够努力。然后到参加飞思卡尔，到数学建模，到考研，到学习，每一步我都是这样想，其实这本身都是没有任何问题的，但我却总是给自己一种暗示:“我是个有着如此多优秀品格的人，只要努力，我就一定能够做到最好”。所以我一直都给自己的期望就是:做最牛逼的人，干最牛逼的事。

可是后来我发现，有些事情的确不是我能够做到的，比如我数学建模的时候能够三天三夜不睡觉，但我的确没有作出什么事情来，说明我真的是不够聪明。做飞思卡尔的时候，半夜调车困得半死，说明我的确精力不够旺盛。过年和小黑他们一起喝酒，喝到现在白酒已经不能沾口，说明我的确不是个能喝酒的人。

我们都是普通人，要知道，人在一定的时间里，是有他上限的。当然，我一直都相信上限是可以通过努力而不断提高的。但是，在很短的时间里，是不会有太大的变化的。所以，不要给自己在某个阶段定下太大的目标，慢慢来嘛。

要先从简单的事情做起才能把事情做好。《失控》里面有一章说到构建生态系统，就是从最简单的生态系统开始，然后一层一层的叠加，以至于达到自然级生态系统的仿真。思维也是一样，实质的思考稳固了，才能想更复杂抽象的东西。而我多年来一直都享受于这种本末倒置的学习状态中不曾感觉到，譬如我上高中的时候做物理难题是个好手，几乎所有的物理题我都能找到正确的解题思路，但我的物理成绩总是很差，因为我数学学的不好，而数学不好的原因是:我讨厌重复的计算，所以我的计算能力很差。舍本逐末的方式让我吃了很多亏。
把复杂的东西简单化，有时候反而能够更清晰的看到事情的本质，这个曾经是我最佩服朱老师的一点，而到了现在，我也似乎也学到了一丝皮毛，然后我就觉得很开心。

我这么多年的长时间的写东西，刚开始的时候是为了装逼给别人看，后来改成发泄自己的情绪，再后来就是到现在，慢慢的在写的时候也会思考一些东西，比如自己哪些地方做的不对，哪些地方需要改进。

过去我给自己订的目标太大了，所以负担就很大，迈不开步子。现在的我有些事情算是想明白了，比如说其实我就是一个普通人，只能做普通的事，但做事情的关键是要让自己开心。突然想到初一的时候看少年文艺里面连载的一篇小说，不记得是叫《男生贾里》还是《女生贾梅》，还是叫什么名字，一个学生说，他们班有个同学，他妈妈冬天给他洗屁股，一下把他的毛线裤和外面的裤子脱光了，然后他很生气，把裤子又穿了上去，然后告诉他妈妈，饭要一口一口吃，裤子要一件一件脱……觉得很有意思。

有时候人被困在某个角落里爬不出来，就得需要别人拉一把，真的很感谢许多朋友和同学问我的那些我没有思考过的问题，还有那个我现在还不知道名字给我打电话鼓励我的同学。

毕业季

2013/4/21

人必须要坦然面对自己。

比如我写了这么久的博客，看的人也就那么几个，我活了二十多年，最关注我的也只有父母，喜欢过那么多的姑娘，却没有见到姑娘说喜欢过我。

我是属于那种从来没有过叛逆期的人，遇到点事情总会压在心里。而青春期的荷尔蒙压抑到一定程度总是需要另外一种渠道去发泄，可能是我不能忍受现实中自己的普通和平凡从而由此而产生的孤独和被漠视，我在网上写的文字都很疯狂，平时突然冒出的话也经常很经常让人感到无法理解，很多同学都说我是个很奇怪的人，大抵是因为我的表现很让人奇怪吧，吓着大家了。

可能还是我这个人比较喜欢逃避问题吧？

仔细想想，其实我就是个很普通的人啊，为什么要给自己安下那么多包袱？为什么会给自己定那么多目标呢？达到了目标又怎么样呢？

无非是再挖一个一个的坑，自己往里面跳而已。

理想是个什么东西？

以前总是会觉得，理想就是做最牛逼的人，干最牛逼的事 。现在我想明白了，理想就是做普通的人，做自己喜欢做的事。

其实，这才是最牛逼的。

在家门前看到通往初中的那条走了无数遍的路，路旁边的树叶都长起来了，真的真的很漂亮，我也很久没有在春天的时候呆在家了。

快毕业了啊，毕业了。不能重新来过，只能从现在开始。

生活永远都是这样，我会想念你们的。
